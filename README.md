The parallel layout in Traktor Pro with 4 decks is ABCD. But the physical layout on the Traktor S4 for example, is CABD.

This python script changes the layout data embedded in the Traktor Pro 4.exe, to change the 4 decks parallell layout from ABCD to CABD.

Useage:
1. Download and extract the repo.
2. Copy the Traktor Pro 4.exe into the extracted folder.
3. Run the python script.
4. Replace the original Traktor Pro 4.exe with the newly created Traktor Pro 4 New.exe

Following these steps should work on any version of Traktor Pro 4.
